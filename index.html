q&aよくある質問

入金とお支払いについて

  • LINE公式アカウントについて
  • LINE公式アカウントでは、お得なキャンペーンやイベントのお知らせなど随時配信しております。入会をご検討の方・会員の方はぜひご登録くださいませ。
  • 入会前に見学できますか？
  • 無料で見学していただけます。お問い合わせ又はお電話でご予約ください。
  • はじめての卓球を利用するにはどうすればいいですか？
  • 当施設は会員制でございます。ご利用には会員申し込みが必要となります。
  • 入会方法について教えてください
  • 本サイトのWEB入会よりご入力いただきます。（5分程度）
    ※WEB入会手続き後にメールが届かないお客様はお問い合わせよりご連絡くださいませ。
  • 初期費用のお支払いについて
  • キャンペーンなどで初期費用無料と記載の場合は無料になります。初期費用（月会費・登録手数料・保険料込）につきましては、以下の方法でお支払いいただけます。
    VISA / MasterCard / JCB / American Express / Diners Club
    ID / QUICPay / 交通系電子マネー / 楽天Edy / nanaco / WAON
  • 使用料・各プランのお支払いについて
  • 口座登録を行い、月1回使用料を口座振替にてお支払いいただきます。

施設のご利用について

  • ルールやマナーはありますか？
  • 入会後、初めての来館時にスタッフが施設・設備の利用方法をご案内いたします。
  • 卓球初心者ですが大丈夫でしょうか？
  • お子様～高齢者、初心者～上級者までレベルに合わせて指導させて頂きます。
  • 子どもは何歳から通えますか？
  • 3才から大丈夫です。身長によっても変わってきますが、通常（76㎝）より10㎝低い高さのバンビ台を一台設置しております。
  • 子どもが練習している間、保護者は見学しても良いですか？
  • 保護者の方が見学される場合は、休憩所をご利用ください。
  • 卓球のラケットやウェアなどがありません
  • お手持ちのTシャツとハーフパンツなどでOKです。また、併設ショップにて卓球用品やウェアなど販売しております。詳細はスタッフにお問い合わせください。
  • 持っていくものを教えてください
  • タオル・着替え・シューズ・ラケット・水分補給用ドリンク・会員証です。併設ショップにて、販売・レンタルサービスがございますので、手ぶらでお越し頂いても大丈夫です。
  • 自分にあった練習法が分かりません
  • 最初はトレーナーにアドバイスを受ける事をおすすめいたします。スキルに合わせたグループレッスンやプライベートレッスンもございます。
  • 更衣室はありますか？
  • 4室ございます。
  • 写真撮影をしても大丈夫ですか？
  • 施設内での撮影はご遠慮くださいませ。試合やイベント時は撮影出来る場合もございますので、詳細はスタッフにお問い合わせください。

併設ショップついて

  • ショップではどんな商品を扱っていますか？
  • シャツ・ソックス・タオル・短パン・シューズ・ラケット・ラバー・ラケットケース・メンテナンス用品・ボール・メンテナンスサービスも行っております。在庫のないものにつきましてはお取り寄せも可能です。また、ドリンクや健康食品なども取り扱っています。
  • ラバーの張替えはできますか？
  • 当店でご購入いただきましたラバーについては、無料で張替えをいたします。
  • レンタル出来るものは何ですか？
  • ラケット200円、シューズ200円、タオル100円は有料レンタルをご利用いただけます。シューズはソックス着用必須となります。
